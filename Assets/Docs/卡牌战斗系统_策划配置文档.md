# 卡牌战斗系统 — 策划配置文档

---

## 一、系统总览

### 1.1 核心玩法

回合制卡牌战斗，玩家通过打出手中卡牌对敌人造成伤害、获得护甲、施加增益。每回合拥有有限行动点，打出卡牌需消耗对应行动点。

### 1.2 战斗流程

```
战斗开始
  │
  ├─ 洗牌（Deck → 抽牌堆）
  ├─ 抽5张牌到手牌
  ├─ 玩家行动点 = 最大行动点（默认3）
  │
  ▼
┌─── 玩家回合 ───────────────────────────┐
│  1. 浏览手牌（鼠标悬浮放大查看）          │
│  2. 点击卡牌打出（消耗行动点）             │
│  3. 可打出多张，直到行动点不足或主动结束    │
│  4. 点击「结束回合」                      │
└──────────────────────────────────────┘
  │
  ├─ 手牌全部弃入弃牌堆
  ├─ 护甲清零（除非本回合打出过坚固卡牌）
  │
  ▼
┌─── 敌人回合 ───────────────────────────┐
│  敌人对玩家造成固定伤害（受玩家护甲减免）  │
└──────────────────────────────────────┘
  │
  ▼
┌─── 新回合开始 ─────────────────────────┐
│  · 行动点回满                            │
│  · 抽5张牌（抽牌堆空了则洗弃牌堆补充）    │
│  · 护甲重置（坚固效果消失，需重新打出）    │
└──────────────────────────────────────┘
  │
  └─ 循环直到一方HP归零
```

### 1.3 胜负条件

| 条件 | 结果 |
|------|------|
| 敌人HP ≤ 0 | 玩家胜利 |
| 玩家HP ≤ 0 | 玩家失败 |

## 二、战斗参数配置

以下参数在 BattleManager 组件的 Inspector 中配置（选中场景中的 `BattleSystem` 对象）。

### 2.1 玩家属性

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Player HP | 50 | 玩家初始生命值 |
| Player Max HP | 50 | 玩家最大生命值 |
| Player Action Points | 3 | 每回合行动点 |
| Player Max Action Points | 3 | 最大行动点上限 |
| Player Armor | 0 | 初始护甲（一般不改） |
| Player Strength | 0 | 初始力量（一般不改） |
| Player Dexterity | 0 | 初始敏捷（一般不改） |

### 2.2 敌人属性

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Enemy HP | 80 | 敌人生命值 |
| Enemy Max HP | 80 | 敌人最大生命值 |
| Enemy Attack Damage | 8 | 敌人每回合造成的伤害 |
| Enemy Name | 敌人 | 显示名称 |

### 2.3 手牌设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Hand Limit | 10 | 手牌上限，超过不抽 |
| Initial Draw | 5 | 战斗开始时抽牌数 |
| Draw Per Turn | 5 | 每回合抽牌数 |

### 2.4 牌组

Deck 列表中拖入卡牌Prefab。同一张卡可拖入多次（表示牌组中有重复卡牌）。

## 三、玩家属性系统

### 3.1 四种属性

| 属性 | 枚举值 | 影响内容 | 初始值 |
|------|--------|----------|--------|
| 力量 | Strength | 攻击牌属性加成伤害 | 0 |
| 敏捷 | Dexterity | 护甲牌属性加成护甲 | 0 |
| 体质 | Vitality | 最大生命值 | 0 |
| 灵巧 | Agility | 每回合抽牌数 | 0 |

### 3.2 属性如何影响卡牌

当卡牌的 Value Type 设为 `AttributeBased` 时：

| 卡牌类型 | 计算公式 | 示例 |
|----------|----------|------|
| 攻击牌 | 基础攻击值 + Attack Attribute对应属性值 | 基础4 + 力量3 = 7点伤害 |
| 护甲牌 | 基础护甲值 + Armor Attribute对应属性值 | 基础4 + 敏捷2 = 6点护甲 |

### 3.3 属性提升途径

通过增益牌的 `IncreaseAttribute` 效果提升。提升后持续生效，影响后续所有相关卡牌。

| 提升属性 | 效果 |
|----------|------|
| 力量+1 | 后续所有属性加成攻击牌额外+1伤害 |
| 敏捷+1 | 后续所有属性加成护甲牌额外+1护甲 |
| 体质+1 | 最大生命值+1，当前生命值同步+1 |
| 灵巧+1 | 每回合抽牌数永久+1 |

## 四、卡牌系统

### 4.1 卡牌三大类型

| 类型 | 中文名 | 模板Prefab | 核心功能 |
|------|--------|-----------|----------|
| Attack | 攻击 | `攻击牌.prefab` | 对敌人造成伤害 |
| Armor | 护甲 | `护甲牌.prefab` | 获得护甲，减免受到的伤害 |
| Buff | 增益 | `增益牌.prefab` | 提升属性、回复资源、抽牌等 |

> 卡牌类型由模板决定，复制模板后创建的卡牌不可更改类型。

### 4.2 卡牌通用属性

所有卡牌都拥有以下属性：

| 字段 | 类型 | 说明 |
|------|------|------|
| Card Name | 文本 | 卡牌名称 |
| Card Art | 图片 | 卡牌插图 |
| Description | 文本 | 描述文案，留空自动生成 |
| Value | 整数 | 商店售价 |
| Grade | 枚举 | 品级 |
| Action Point Cost | 整数 | 行动点消耗 |
| Consume Type | 枚举 | 消耗类型 |
| Keywords | 多选 | 词条 |

#### 品级

| 枚举值 | 中文 | 边框颜色 | 定位 |
|--------|------|----------|------|
| Common | 普通 | 灰色 | 基础卡牌 |
| Fine | 优秀 | 绿色 | 略强于基础 |
| Rare | 精良 | 蓝色 | 中坚力量 |
| Epic | 史诗 | 紫色 | 强力卡牌 |
| Legendary | 传说 | 金色 | 顶级卡牌 |

#### 消耗类型

| 枚举值 | 中文 | 打出后去向 | 适用场景 |
|--------|------|-----------|----------|
| None | 不消耗 | 进入弃牌堆，洗牌后可再抽到 | 普通卡牌 |
| ThisBattle | 本次战斗消耗 | 永久移除，本场战斗不再出现 | 一次性强力卡 |
| ThisRun | 本局游戏消耗 | 永久移除，本局游戏不再出现 | 超强力一次性卡 |

#### 词条（9种）

词条可多选，一个卡牌可同时拥有多个词条。

| 枚举值 | 词条名 | 效果 | 适用类型 |
|--------|--------|------|----------|
| Pierce | 穿透 | 攻击无视敌人护甲 | 攻击牌 |
| Lifesteal | 吸血 | 回复造成伤害的50%生命值 | 攻击牌 |
| Combo | 连击 | 攻击次数+1 | 攻击牌 |
| Heavy | 重击 | 25%概率造成双倍伤害 | 攻击牌 |
| Swift | 迅捷 | 打出后额外抽1张牌 | 攻击牌 |
| Toxic | 剧毒 | 额外造成2点伤害（无视护甲） | 攻击牌 |
| Burning | 燃烧 | 额外造成3点伤害（无视护甲） | 攻击牌 |
| Sturdy | 坚固 | 获得的护甲不随回合消失 | 护甲牌 |
| FreePlay | 免费 | 打出不消耗行动点 | 全部 |

> **配错不会报错但无效果**：如给护甲牌配穿透词条不会生效。

## 五、攻击牌规则

### 5.1 攻击牌专属属性

| 字段 | 类型 | 说明 |
|------|------|------|
| Attack Count | 整数 | 攻击次数 |
| Attack Value Type | 枚举 | 数值计算方式 |
| Attack Value | 整数 | 基础攻击数值 |
| Attack Attribute | 枚举 | 属性加成时使用的玩家属性 |
| Ignore Armor | 勾选 | 是否无视敌人护甲 |

### 5.2 伤害计算公式

```
单次伤害 = Attack Value + (Attack Attribute对应属性值, 仅AttributeBased模式)

总伤害 = 单次伤害 × 攻击次数

如果勾选Combo词条: 攻击次数 = 配置值 + 1
如果勾选Toxic词条: 总伤害 += 2（无视护甲）
如果勾选Burning词条: 总伤害 += 3（无视护甲）
如果勾选Heavy词条: 每次攻击25%概率伤害×2
```

### 5.3 护甲抵消规则

| 情况 | 规则 |
|------|------|
| 正常攻击 | 先扣敌人护甲，护甲扣完后剩余伤害扣血 |
| Ignore Armor = true | 无视护甲，直接扣血 |
| 勾选Pierce词条 | 无视护甲，直接扣血 |
| 勾选Toxic/Burning词条的额外伤害 | 无视护甲，直接扣血 |

### 5.4 吸血规则

勾选Lifesteal词条时，回复量 = 实际造成伤害 × 50%（向下取整）。只计算实际造成的伤害（被护甲抵消的部分不计入）。

### 5.5 攻击牌配置示例

**AK47**：2费，10点固定伤害，穿透

```
Attack Value Type:  Fixed
Attack Value:       10
Attack Count:       1
Keywords:           Pierce
```

**军体拳**：1费，3点×3次，连击（实际4次）

```
Attack Value Type:  Fixed
Attack Value:       3
Attack Count:       3
Keywords:           Combo  → 实际打4次，共12点
```

**属性加成攻击牌**：1费，基础5 + 力量值

```
Attack Value Type:  AttributeBased
Attack Value:       5
Attack Attribute:   Strength
→ 玩家力量=0时伤害5，力量=3时伤害8
```

## 六、护甲牌规则

### 6.1 护甲牌专属属性

| 字段 | 类型 | 说明 |
|------|------|------|
| Armor Value Type | 枚举 | 数值计算方式 |
| Armor Value | 整数 | 基础护甲值 |
| Armor Attribute | 枚举 | 属性加成时使用的玩家属性 |

### 6.2 护甲计算公式

```
获得护甲 = Armor Value + (Armor Attribute对应属性值, 仅AttributeBased模式)

如果勾选Sturdy词条: 本回合获得的护甲不会在回合结束时清零
```

### 6.3 护甲结算规则

| 时机 | 规则 |
|------|------|
| 受到攻击时 | 护甲优先抵消伤害，护甲扣完后剩余伤害扣血 |
| 回合结束时 | 护甲清零（除非本回合打出过坚固卡牌） |
| 坚固效果 | 仅持续到下个回合结束，需再次打出坚固卡牌续期 |

### 6.4 护甲牌配置示例

**防弹衣**：1费，6点固定护甲，坚固

```
Armor Value Type: Fixed
Armor Value:      6
Keywords:         Sturdy
→ 获得6护甲，回合结束不清零，持续到下回合结束
```

**迷彩服**：1费，4 + 敏捷值护甲

```
Armor Value Type: AttributeBased
Armor Value:      4
Armor Attribute:  Dexterity
→ 敏捷=0时护甲4，敏捷=2时护甲6
```

## 七、增益牌规则

### 7.1 增益牌专属属性

| 字段 | 类型 | 说明 |
|------|------|------|
| Buff Duration | 枚举 | 增益时效 |
| Buff Duration Turns | 整数 | 回合数（仅BattleXTurns模式显示） |
| Buff Stacks | 整数 | 增益层数 |
| Buff Effects | 列表 | 效果列表，可添加多个 |

### 7.2 增益时效

| 枚举值 | 中文 | 效果 |
|--------|------|------|
| GlobalPermanent | 全局永久 | 永久生效，跨战斗保留 |
| BattlePermanent | 局内永久 | 本场战斗内永久生效 |
| BattleXTurns | 局内x回合 | 生效指定回合数 |

### 7.3 增益效果类型

每张增益牌可包含多个效果，效果数值受层数影响：

```
实际效果值 = 单个效果Value × Buff Stacks
```

| 效果类型 | 枚举值 | 说明 | 额外字段 |
|----------|--------|------|----------|
| 提升属性 | IncreaseAttribute | 永久提升指定属性 | Target Attribute（选择属性） |
| 回复行动力 | RestoreActionPoints | 立即回复行动点 | 无 |
| 抽牌 | DrawCards | 立即从抽牌堆抽牌 | 无 |
| 获得护甲 | GainArmor | 立即获得护甲 | 无 |
| 回复生命 | HealHP | 立即回复生命值 | 无 |

### 7.4 增益牌配置示例

**力量提升**：1费，力量+2，局内永久

```
Buff Duration:  BattlePermanent
Buff Stacks:    1
Buff Effects:   [1个]
  → Effect Type:      IncreaseAttribute
  → Value:            2
  → Target Attribute: Strength
```

**双重增益牌**：1费，抽2张牌 + 获得5护甲

```
Buff Duration:  BattlePermanent
Buff Stacks:    1
Buff Effects:   [2个]
  ① Effect Type:  DrawCards, Value: 2
  ② Effect Type:  GainArmor, Value: 5
```

**多层增益牌**：2费，力量+1×3层 = 力量+3

```
Buff Duration:  BattlePermanent
Buff Stacks:    3
Buff Effects:   [1个]
  → Effect Type:      IncreaseAttribute
  → Value:            1
  → Target Attribute: Strength
  → 实际提升: 1 × 3 = 力量+3
```

## 八、牌堆机制

### 8.1 三个牌堆

| 牌堆 | 说明 |
|------|------|
| 抽牌堆 | 未抽到的卡牌，按顺序抽取 |
| 手牌 | 当前可打出的卡牌，最多10张 |
| 弃牌堆 | 已打出的不消耗卡牌和回合结束时弃掉的手牌 |

### 8.2 洗牌规则

```
战斗开始: Deck → 洗牌 → 抽牌堆
抽牌时: 抽牌堆空 → 弃牌堆 → 洗牌 → 抽牌堆
打出卡牌: 不消耗 → 弃牌堆
          本战消耗 → 移除池（不再进入任何牌堆）
```

### 8.3 回合结束弃牌

玩家点「结束回合」时，手中所有卡牌弃入弃牌堆（不会消耗），下回合重新抽5张。

## 九、新建卡牌教学

### 第1步：选择模板

```
Assets/Prefabs/Battle/Cards/
├── 攻击牌.prefab   ← 制作攻击卡牌复制此模板
├── 护甲牌.prefab   ← 制作护甲卡牌复制此模板
└── 增益牌.prefab   ← 制作增益卡牌复制此模板
```

选中对应模板，按 **Ctrl+D**（Mac: **Cmd+D**）复制。

### 第2步：重命名

重命名为卡牌名，如 `手雷.prefab`。

### 第3步：配置Inspector

选中Prefab，在Inspector的 `Card Display` 组件中配置。Inspector只显示当前类型相关的字段。

### 第4步：加入牌组

打开 `Assets/Scenes/Battle.scene` → Hierarchy选中 `BattleSystem` → `Battle Manager` 的 `Deck` 列表 → 点 `+` → 拖入新卡牌Prefab。

### 第5步：测试

点击运行按钮进入Play Mode，验证卡牌效果。

## 十一、设计参考数值

### 11.1 行动点经济

| 设计原则 | 说明 |
|----------|------|
| 每回合3行动点 | 1张3费卡 或 1张2费+1张1费 或 3张1费 或 1张1费+1张0费 |
| 0费卡 | 应设计为较弱效果（如2点伤害），配合迅捷词条有额外价值 |
| 高费卡（2-3费） | 应为高伤害或强力效果，需权衡是否值得一回合只打一张 |

### 11.2 伤害参考

| 卡牌定位 | 建议总伤害 | 费用参考 |
|----------|-----------|----------|
| 基础攻击 | 5-8 | 1费 |
| 中阶攻击 | 9-14 | 2费 |
| 高阶攻击 | 15-20 | 3费或带强力词条 |

### 11.3 护甲参考

| 卡牌定位 | 建议护甲值 | 费用参考 |
|----------|-----------|----------|
| 基础护甲 | 4-6 | 1费 |
| 中阶护甲 | 7-10 | 2费 |
| 带坚固词条 | 基础值-1~2 | 1-2费 |

### 11.4 敌人血量参考

| 敌人类型 | 建议HP | 说明 |
|----------|--------|------|
| 普通敌人 | 30-60 | 3-5回合可解决 |
| 精英敌人 | 80-120 | 需要配合增益卡牌 |
| Boss | 150-300 | 长期战斗，需要续航能力 |
